# рџ’і Machine Learning-Based Credit Card Approval Prediction System

## Overview

The **Machine Learning-Based Credit Card Approval Prediction System** is a web-based intelligent application that predicts whether a credit card application is likely to be approved based on an applicant's financial and demographic information.

The system leverages supervised Machine Learning algorithms to automate the credit card approval process, helping financial institutions make faster, more consistent, and data-driven decisions while reducing manual effort.

The application is built using **Python**, **Flask**, **Scikit-learn**, and **MySQL**, providing an intuitive web interface for applicants and administrators.

# Project Objectives

- Predict credit card approval using Machine Learning.
- Reduce manual verification time.
- Improve prediction accuracy.
- Provide secure user authentication.
- Store applicant information securely.
- Generate real-time approval predictions.
- Maintain prediction history.
- Support future model improvements.

# Machine Learning Algorithms

The project evaluates multiple classification algorithms:

- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

The best-performing model is used for prediction.
